# Anime Search App 🎌

A simple anime search web app built with React + TypeScript using Jikan API.

## 🚀 Features
- Search anime by title
- Pagination (desktop & mobile)
- Anime detail page
- Trailer modal
- Dark mode support
- Responsive UI

## 🛠 Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router
- Zustand (state management)

## 📡 API
Powered by Jikan API (MyAnimeList unofficial API)
